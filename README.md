# AI-Based Health Monitoring System

## 📖 Project Overview
The AI-Based Health Monitoring System is a web application that helps users monitor their health by recording and analyzing health-related data. The system uses Machine Learning techniques to provide intelligent insights and predictions.

## ✨ Features
- User Registration and Login
- Health Data Management
- AI-Based Health Analysis
- Secure Database Storage
- User-Friendly Interface

## 🛠 Technologies Used
- Python
- Django
- HTML
- CSS
- JavaScript
- SQLite
- Machine Learning

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/kusumanjali335-create/AI-Based-Health-Monitoring-System.git
```

2. Create a virtual environment

```bash
python -m venv venv
```

3. Activate the virtual environment

Windows:

```bash
venv\Scripts\activate
```

4. Install dependencies

```bash
pip install -r requirements.txt
```

5. Run the server

```bash
python manage.py runserver
```

6. Open your browser

```
http://127.0.0.1:8000/
```
